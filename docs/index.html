Overview

Flux provides a seamless experience for browsing and watching movies and TV shows. It leverages the power of the TMDB API for rich metadata and integrates with the Vidsrc API to fetch streamable video links. The application is entirely client-side, offering a user-friendly interface.

Warning: This application is developed for personal use and educational purposes. It accesses potentially copyrighted content via third-party APIs and will not be published on official app stores.
